Instrucciones de EL RETO

Este Reto lo propuse para la lista de programación de C/C++ CWORLD. Si desea ver los demás retos o suscribirse a la lista puede visitar:

 http://www.ciudadfutura.com/cworld/elreto.htm

Para que sea mas interesante le sugiero que primero intente resolverlo por sus propios medios. Y solo cuando ya no pueda seguir, mirar la solución.

Archivos Incluidos:

retoxxxx.*    -> Enunciado Original
juezxxxx.txt  -> Solución JUEZ ( la solución que utilicé
                  para corregir a los participantes )
juezxxxxax.*  -> Código Fuente Adjunto de la
                  solución JUEZ ( opcional )
soluxxxx.c    -> Solución enviada por el participante
                  ganador ( no siempre existe )
discuxxxx.txt -> Algunas discusiones importantes surgidas
                  durante el desarrollo del reto (opcional)
